วิธีใช้ HDDCAT

จากแตก zip ถึงค้นเจอทุกไฟล์ — ใช้เวลาไม่ถึง 5 นาที

1

ดาวน์โหลดแล้วแตก zip

กดปุ่ม Download Free แล้วแตกไฟล์ HDDCAT.zip — ข้างในมี HDDCAT.app ลากไปไว้โฟลเดอร์ Applications ได้เลย

📦 แตก HDDCAT.zip แล้วจะเจอ:
HDDCAT.appตัวแอป — ดับเบิลคลิกเปิดใช้ได้เลย ลากไป Applications ได้
🐍
catalog.pyตัวเดียวกับในแอป สำหรับสาย Terminal/CLI
📄
README.mdคู่มือฉบับย่อ ภาษาไทย
⚖️
LICENSEMIT — ฟรี ใช้/แก้/แจกต่อได้
2

เปิดครั้งแรก: คลิกขวา > Open

คลิกขวาที่ HDDCAT.app แล้วเลือก Open — ครั้งแรกครั้งเดียว macOS จะถามยืนยันเพราะแอปมาจากอินเทอร์เน็ต จากนั้นเบราว์เซอร์จะเปิด HDDCAT ขึ้นมาเอง ไอคอนแมวจะอยู่ใน Dock — อยากปิดเมื่อไหร่ คลิกขวาที่ไอคอนแล้ว Quit · ถ้าเครื่องยังไม่มี python3 ระบบจะเด้งชวนติดตั้ง Command Line Tools ให้กด Install

3

สแกนไดรฟ์ลูกแรก

เสียบ HDD แล้ว HDDCAT จะเด้งถามเองว่า "สแกนเลยไหม" — หรือไปที่แท็บสแกน ใส่ path ของไดรฟ์กับตั้งชื่อ label · ทิป: ตั้ง label ให้ตรงกับสติกเกอร์ที่แปะบนไดรฟ์จริง เวลาค้นเจอจะได้หยิบถูกลูกทันที

4

ค้นทุกอย่างจากที่เดียว

แท็บคลังงานรวมทุกไดรฟ์ไว้ในตารางเดียว — พิมพ์ค้นชื่อลูกค้า ชื่องาน หรือชื่อไฟล์ กรองตามไดรฟ์หรือประเภทงาน คลิกที่แถวเพื่อดูไฟล์ข้างในได้เลย และทั้งหมดนี้ทำได้แม้ไดรฟ์จะไม่ได้เสียบอยู่ — ระบบบอกว่าไฟล์อยู่ไดรฟ์ลูกไหน โฟลเดอร์ไหน

5

หาไฟล์ซ้ำ คืนพื้นที่

แท็บไฟล์ซ้ำ เลือกขนาดขั้นต่ำแล้วกดค้นหา — ระบบเทียบชื่อและขนาดไฟล์ข้ามทุกไดรฟ์ แล้วบอกว่าคืนพื้นที่ได้เท่าไหร่ โหมดอ่านอย่างเดียว ไม่มีการลบไฟล์อัตโนมัติ คุณตัดสินใจเองทุกไฟล์

เกร็ดที่ควรรู้

  • เปลี่ยนชื่อไดรฟ์แล้วขึ้นซ้ำเป็น 2 ลูก? ไปที่แท็บไดรฟ์ แล้วกด "ลบออกจาก catalog" ที่ลูกเก่า — ไฟล์จริงบนไดรฟ์ไม่ถูกแตะ
  • ข้อมูลทั้งหมดอยู่ในไฟล์ catalog.db ข้างตัวโปรแกรม — ย้ายเครื่องแค่ก๊อปทั้งโฟลเดอร์ไป ไม่มีอะไรอยู่บน cloud
  • สาย Terminal ใช้ CLI ได้เต็มรูปแบบ: python3 catalog.py scan /Volumes/ไดรฟ์ --label ชื่อ · search คำค้น · dedup · serve
